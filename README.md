## 🎯 평가 기준 (MEC - Model Evaluation Criteria)

### 5가지 평가 축

| 축 | 정의 | 배점 | 경량 모델 고려사항 |
|---|------|------|---------|
| **정확성** | 원문 사실 반영도 | 25점 | 세부 정보 누락 가능성 |
| **구조화** | 논리적 정렬 수준 | 20점 | 복잡한 구조 생성 어려움 |
| **가독성** | 이해도 및 명확성 | 20점 | 간결성에서 우수 가능 |
| **불명확성 처리** | "모른다" 명시도 | 20점 | 안전장치 약할 수 있음 |
| **응답 속도** | 실시간 처리 능력 | 15점 | **경량 모델의 주요 강점** |

**총점**: 100점

## 🔑 주요 발견사항

### 1️⃣ Claude Haiku 4.5
**"속도의 왕"**

✅ **강점**
- 가장 빠른 응답 속도 (2.3초)
- 간결한 출력으로 정보 과부하 없음
- 무료 버전 일일 50회 요청 가능

❌ **약점**
- 정보 누락 (마술 공연 누락)
- 불명확성 처리 미흡
- 시각적 요소(이모지) 부족

📌 **추천 사용 시나리오**
```
✓ 실시간 채팅봇
✓ 대량 배치 처리 (속도 우선)
✓ 모바일 앱 통합 (응답 시간 중요)
✗ 정확성이 중요한 보고서 작성
```

---

### 2️⃣ GPT-5.4 mini
**"균형잡힌 선택"** ⭐ 최종 추천

✅ **강점**
- 정확성과 속도의 최고 균형 (89점)
- 명확한 구조화 (섹션 분리)
- 적절한 이모지 사용으로 가독성 우수
- 불명확한 부분 명시 (신뢰도 높음)

❌ **약점**
- 응답 속도 중간 수준 (3.1초)
- 일부 정보 중복 배치

📌 **추천 사용 시나리오**
```
✓ 회의록 요약 및 안내문 작성 (최적)
✓ 보고서 초안 생성
✓ 주민/고객 대상 공지문
✓ 정확성과 속도 모두 필요한 업무
```

---

### 3️⃣ Gemini 3 Flash
**"상세 정보 제공자"** ⚠️ 환각 주의

✅ **강점**
- 빠른 응답 속도 (2.7초)
- 다양한 이모지로 시각적 매력
- 상세한 설명 제공

❌ **약점**
- **환각(Hallucination) 위험** ⚠️
  - 원문에 없는 "쿠폰 내용" 구체화
  - 불명확한 부분을 임의로 채우는 경향
- 불명확성 처리 약함 (14/20)
- 사실 검증 필요

📌 **추천 사용 시나리오**
```
✓ 창의적 콘텐츠 생성 (소설, 마케팅 카피)
✓ 빠른 초안 생성 (사실 검증 필수)
✗ 사실 기반 보고서 (환각 위험)
✗ 공식 공지문 (신뢰도 중요)
```

---

## 🧪 환각(Hallucination) 검증 결과

### 테스트 케이스: "축제 쿠폰 내용"

**원문**: "다음 행사 쿠폰을 드립니다. (구체적 내용 미정)"

| 모델 | 결과 | 판정 |
|------|------|------|
| Claude Haiku | "추후 안내" 명시 | ✅ PASS |
| GPT-5.4 mini | "구체적 내용 미정" 표기 | ✅ PASS |
| Gemini 3 Flash | "10% 할인 쿠폰" 구체화 | ❌ FAIL (환각) |

**결론**: Gemini는 불확실한 정보를 임의로 구체화하는 경향 → 사실 검증 필수

## 📋 사용 방법

### 1️⃣ 기본 프롬프트 사용

```python
# system_prompt_v2.txt 참고
system_prompt = """
당신은 마을 자치 회장의 보좌관입니다.
복잡한 회의 내용을 주민 대상 안내문으로 변환하는 역할을 합니다.

[요구사항]
1. 회의 원문을 주민 친화적 안내문으로 변환
2. 불명확한 부분은 "추후 안내" 표기
3. 이모지와 섹션 구분으로 가독성 강화
"""

user_input = """
[목적] 축제 안내문 작성
[대상] 마을 주민
[톤] 친절하고 긍정적
[금지어] 전문용어, 복잡한 표현
"""
```

### 2️⃣ 입력 템플릿

```markdown
[목적] 
[대상] 
[핵심 포인트] 
[톤] 
[금지어] 
[필수 포함 항목] 
[원문]
```

### 3️⃣ 검증 체크리스트

- [ ] 원문의 모든 결정사항 포함?
- [ ] 불명확한 부분 명시?
- [ ] 주민 이해 수준에 맞는 언어?
- [ ] 이모지/섹션 구분 적절?
- [ ] 사실 오류 없음?
